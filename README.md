# CineBuddy - Private Movie Night

Watch movies in sync with your friend. 2-person private rooms, YouTube, local videos, video/voice call, chat.

## Run on Replit (Free)

1. Upload all files to Replit
2. Click the green Run button
3. Share your Replit URL with your friend
4. Create a room and enjoy!

## Run Locally

```
pip install -r requirements.txt
python app.py
```

Open http://localhost:8080
